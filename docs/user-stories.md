# User Stories

> This document captures planned user stories for ElderCare-Invest, grouped by feature area. More stories will be added as requirements are refined.

## Authentication & Profile

As a user,
I want to create an account,
So that I can start planning my future elder care.

As a user,
I want to log in and log out securely,
So that my financial information stays protected.

As a user,
I want to manage my profile details,
So that my information stays accurate and up to date.

## Investments

As a user,
I want to invest monthly,
So that I can build my elder care fund over time.

As a user,
I want to view my investment portfolio,
So that I can track my progress toward my elder care goals.

As a user,
I want to use a retirement savings calculator,
So that I can estimate how much I need to save for elder care.

## Healthcare

As a user,
I want to add and manage my healthcare plans,
So that I can keep my care coverage organized in one place.

## Elder Care Marketplace

As a user,
I want to browse elder care facilities,
So that I can compare options and plan ahead.

As a family member,
I want to explore elder care options on behalf of a parent,
So that I can help plan for their future care needs.

## Insurance

As a user,
I want to link my insurance policy to my account,
So that I can manage all my care-related plans together.

## Payments

As a user,
I want to make a one-time or recurring payment toward my investment plan,
So that I can consistently grow my elder care fund.

## Reports & Notifications

As a user,
I want to receive reminders about upcoming contributions,
So that I don't miss a scheduled investment.

As a user,
I want to view financial reports on my investments,
So that I understand how my elder care fund is growing.

## Document Storage

As a user,
I want to securely store important financial and medical documents,
So that I have them accessible in one place when needed.

## Admin

As an admin,
I want to manage registered users and partnered providers,
So that the platform stays accurate and trustworthy.

## Future (AI Recommendations)

As a user,
I want to receive personalized financial recommendations,
So that I can make better decisions about my elder care planning.
